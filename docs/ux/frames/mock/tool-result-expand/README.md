# tool-result-expand

도구 결과가 접힌 상태에서 펼치는 경험을 검증한다. FP-10(접힌 상태 가시성 없음), FP-04(Ctrl+O 키바인딩 미노출), FP-13(CodeView 80줄 제한)을 재현한다.

| # | 단계 | 상태 | 오류 |
|---|------|------|------|
| 1 | 초기 idle 화면 | ok |  |
| 2 | 도구 결과 1개 주입 — 120줄 file_read | ok |  |
| 3 | 접힌 상태 — 펼침 키 안내가 보이는가? | assertion-failed | assertion failed at "접힌 상태 — 펼침 키 안내가 보이는가?" |
| 4 | Ctrl+O로 펼치기 | ok |  |
| 5 | 펼친 상태 — 내용이 보이고 80줄 제한 표시가 있는가? | ok |  |
| 6 | Ctrl+O로 다시 접기 | ok |  |
| 7 | 접힌 상태 복귀 프레임 | ok |  |

각 단계의 프레임은 같은 디렉토리의 `NN-*.txt` 파일에 저장되어 있습니다.
