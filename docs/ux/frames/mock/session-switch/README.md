# session-switch

멀티세션을 생성하고 전환할 때, 현재 세션이 StatusBar에 표시되는지 확인한다. FP-14 수정 이후 StatusBar 기본 항목에 session이 포함되어 전환 후 세션명이 즉시 보여야 한다.

**결과**: 5/5 단계 통과

| # | 단계 | 상태 | 오류 |
|---|------|------|------|
| 1 | 초기 화면 | ok |  |
| 2 | 세션 목록 조회 | ok |  |
| 3 | 새 세션 work 생성 | ok |  |
| 4 | 세션 work로 전환 요청 | ok |  |
| 5 | 전환 후 화면 — 현재 세션 가시성 + 성공 피드백 | ok |  |

각 단계의 프레임은 같은 디렉토리의 `NN-*.txt` 파일에 저장되어 있습니다.
