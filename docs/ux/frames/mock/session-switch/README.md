# session-switch

멀티세션을 생성하고 전환할 때, 현재 세션이 StatusBar 또는 화면 어딘가에 표시되는지 확인한다. 현재는 StatusBar에 session 항목이 없으므로 전환 후에도 화면에 세션명이 나타나지 않을 것으로 예상된다. 이 시나리오는 해당 UX 공백을 재현하기 위한 것이다.

**결과**: 4/5 단계 통과

| # | 단계 | 상태 | 오류 |
|---|------|------|------|
| 1 | 초기 화면 | ok |  |
| 2 | 세션 목록 조회 | ok |  |
| 3 | 새 세션 work 생성 | ok |  |
| 4 | 세션 work로 전환 요청 | ok |  |
| 5 | 전환 후 화면 — 현재 세션 가시성 | assertion-failed | assertion failed at "전환 후 화면 — 현재 세션 가시성" |

각 단계의 프레임은 같은 디렉토리의 `NN-*.txt` 파일에 저장되어 있습니다.
